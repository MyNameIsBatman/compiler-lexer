programa    : declaração*

declaração  : expressao (EOL|EOF)

expressao   : VAR|CONST IDENTIFIER EQ expressao
            : termo ((MAIS|MENOS) termo)*

termo       : fator ((MULTIPLICAR|DIVIDIR) fator)*

fator       : (MAIS|MENOS) fator
            : power

power       : call (POWER fator)*

call        : atom (LPAREN (expressão (COMMA expressão)*)? RPAREN)?

atom        : INT|FLOAT|STRING|IDENTIFIER
            : LPAREN expressão RPAREN
            : expressão_if

expressão_if: IF LPAREN expressão RPAREN LBRACE expressão RBRACE
                (ELSEIF LPAREN expressão RPAREN LBRACE expressão RBRACE)*
                (ELSE expressão)?
